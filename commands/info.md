### The info commands.

The info commands give statistical information on the bot, server and specific users.

| **Usage** | **Permissions** | **Description** |
| :--- | :--- | :--- |
| !!serverinfo | None | Give server information e.g owner,     member count, region etc. |
| !!botinfo | None | Gives bot information such as a link tothe github repo, guild size and a link tomy website. |
| !!userinfo \[user\]                                      _Alias !!ui \[user\]_ | None | Gives user information e.g time on the server, time since last message etc. |
